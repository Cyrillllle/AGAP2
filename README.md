📄 README
🚀 Installation
Aller dans le dossier resources
Lancer le fichier .bat pour installer automatiquement les dépendances :
install.bat
▶️ Lancement de l'application

Une fois les dépendances installées, lancer l'application Streamlit avec la commande :

streamlit run main.py
📦 Prérequis
Python installé (version 3.x recommandée)
pip disponible
⚠️ Remarques
Le script .bat installe automatiquement toutes les librairies nécessaires
Assurez-vous d’exécuter la commande dans le bon dossier (racine du projet)# AGAP2
