Installation\n

Aller dans le dossier resources\n
Lancer le fichier .bat pour installer automatiquement les dépendances :\n
install.bat


Lancement de l'application\n

Une fois les dépendances installées, lancer l'application Streamlit avec la commande :\n
streamlit run main.py


Prérequis\n

Python installé (version 3.14 recommandée)\n
pip disponible


Remarques\n

Le script .bat installe automatiquement toutes les librairies nécessaires\n
Assurez-vous d’exécuter la commande dans le bon dossier (racine du projet)# AGAP2
